== Description ==

* Switch the theme by the user agent.
* Can be specified user agent by a regular expression.
* It is possible to register a maximum seven devices.


Translators

*   Japanese (ja) - [Katsushi Kawamori](http://gallerylink.nyanko.org/)

== Installation ==

1. Upload `useragent-themes-switcher` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the Settings page, set the theme and the corresponding user agent.
